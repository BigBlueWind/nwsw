Commands

The operator can access various eNodeB/Base Station commands in this section towards carrying-out various management based activities. Operator is provided with set of commands in command tree window for initiating file downloads (e.g. software), file uploads (e.g. logs), software install, etc. for the connected eNodeB.

For some commands (e.g. Software Activation, Base Station Reboot, Hardware Administration, etc.) it's a requisite that eNodeB should be in LOCKED state, such parameters are disabled for editing in case eNodeB is in UNLOCKED state.



Following commands are supported.

- Download/Upload files to/from eNodeB
- Reboot/Shutdown entire Base Station
- Change the administration state (Lock/Unlock) of eNodeB
- Software Installation and Activation for eNodeB
- Hardware Administration (reset Radio Head/WiFi/GPS)
- Start/Stop Tracing the protocol messages (RRC/S1AP/MAC)
- Cell Administration
- Counter Control
